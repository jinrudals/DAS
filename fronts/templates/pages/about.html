Design Automation System

The Design Automation System (DAS) is a comprehensive platform for design workflow automation, regression testing management, and verification infrastructure for semiconductor design teams.

Key Features

  • Automated Regression Testing - Continuous verification of design changes
  • Project Management - Centralized tracking of design projects and targets
  • Jenkins Integration - Seamless CI/CD pipeline integration for automated workflows
  • Real-time Monitoring - Live status tracking and execution monitoring
  • Advanced Filtering - Powerful search and filter capabilities for large datasets

Technology Stack

Backend: Django with PostgreSQL
Frontend: Bootstrap 5 with modern JavaScript
CI/CD: Jenkins integration
Authentication: Django user management

System Architecture

DAS follows a modular architecture with separate applications for different concerns:

  • Modeling: Core data models for projects, targets, criteria, and executions
  • Fronts: User interface and web presentation layer
  • Accounts: User authentication and profile management

Contact & Support
